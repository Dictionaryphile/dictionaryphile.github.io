## 一些金玉良言
- 使用才是真正的占有。
- 查词典既是找答案，也是找问题。
- 查词典可以解决绝大部分语言方面的问题。
- 一本牛津高阶，顶一百个英语名师。
- 老师就是比你先查了词典而已。
- 知识越多的人，查词典越多。
- 不查词典，不学习的人，是可耻的。
- 其实不管学什么，最终都是把自己变成一本相关领域的活词典，仅此而已。
- 一切问题，都是语言和逻辑的问题，当然，还可能是，钱的问题。
- Once you’ve read the dictionary, every other book is just a remix.
